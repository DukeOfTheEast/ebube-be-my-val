## Setup Instructions

1. Make sure you have Node.js installed (v18 or higher recommended)

2. Navigate to the project directory:
   ```bash
   cd valentine-app
   ```

3. Install dependencies:
   ```bash
   npm install
   ```

4. Run the development server:
   ```bash
   npm run dev
   ```

5. Open your browser and visit: http://localhost:3000

## Building for Production

To create an optimized production build:

```bash
npm run build
npm start
```

## Deployment

You can easily deploy this to:
- **Vercel** (recommended for Next.js): Just connect your GitHub repo
- **Netlify**: Deploy from GitHub with auto-detect settings
- **Any static host**: Run `npm run build` and deploy the `out` folder
